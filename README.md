# ⚡ Клодування 3D — Telegram Bot + Web Dashboard

## Структура проєкту
```
intensive_node/
├── index.js        ← точка входу (запускає бот + сервер)
├── bot.js          ← Telegram бот
├── server.js       ← Express веб-сервер + API
├── db.js           ← JSON база даних
├── package.json
├── data/
│   └── db.json     ← всі дані (автоствоюється)
└── public/
    ├── index.html  ← красивий лендінг
    ├── login.html  ← вхід в адмін-панель
    └── admin.html  ← повна адмін-панель
```

## Встановлення

```bash
cd intensive_node
npm install
```

## Запуск

```bash
node index.js
```

або для розробки (авто-перезапуск):
```bash
npm run dev
```

## Налаштування

Відкрийте `bot.js` та `server.js` і замініть:
```
BOT_TOKEN     = "ВАШ_ТОКЕН_BOTFATHER"
ADMIN_ID      = ВАШ_TELEGRAM_ID
ADMIN_PASSWORD = "ваш_пароль_адміна"
```

Або через змінні середовища:
```bash
BOT_TOKEN=... ADMIN_ID=... ADMIN_PASSWORD=... node index.js
```

## Адмін-панель

Сайт: http://localhost:3000  
Адмін: http://localhost:3000/admin

## Функції

### 🤖 Telegram Бот
| Команда | Дія |
|---|---|
| `/start` | Лендінг курсу або перегляд відео |
| `/help` | Список команд |
| `/setup` | Налаштувати курс (діалог) |
| `/add_video` | Завантажити відео |
| `/del_video <№>` | Видалити відео |
| `/grant <id>` | Видати доступ |
| `/revoke <id>` | Відкликати доступ |
| `/buyers` | Список покупців |
| `/pending` | Список заявок |
| `/broadcast <текст>` | Розсилка всім |
| `/stats` | Статистика в боті |

### 🌐 Адмін-панель сайту
- 📊 **Дашборд** — 8 ключових метрик, графіки активності, топ відео, типи подій
- 👥 **Покупці** — таблиця, відкликання доступу
- ⏳ **Заявки** — видача/відхилення одним кліком (бот сповіщає користувача)
- 🎬 **Відео** — список з видаленням
- ✏️ **Курс** — редагування назви, опису, ціни
- 📢 **Розсилка** — текстова розсилка всім покупцям через бот
- 📈 **Статистика** — повний лог всіх подій бота та відвідувань сайту
- 💾 **Експорт/Імпорт** — ZIP архів / JSON / CSV статистики; імпорт з merge

### 🔒 Захист відео
- `protect_content: true` — блокує пересилання та збереження в Telegram
- Відео зберігаються як `file_id` на серверах Telegram
- Доступ тільки через кнопки навігації в боті
